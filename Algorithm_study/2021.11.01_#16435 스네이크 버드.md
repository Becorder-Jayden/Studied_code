## 16435 스네이크 버드

<br/>

풀이

```python
N, L =  list(map(int, input().split())) # 과일개수 N, 스네이크 버드 초기 길이 L
h = list(map(int, input().split())) # 과일 높이
h = sorted(h)
# h.sort()으로 변환하면 채점에서 틀렸다고 함

for i in range(N):
    if h[i] <= L:
        L+=1
print(L)
```

<br/>

배운 점

* 정렬할 때에는 변수를 만들어주어야 한다. sort()로 작성하면 오답으로 채점
* 간단한 풀인데 머리가 굳어있는걸까.. 논리적으로 풀어가는 연습이 더 필요할 듯 하다
