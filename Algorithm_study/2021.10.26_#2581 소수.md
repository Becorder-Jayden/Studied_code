## #2581 소수_런타임 에러 미해결

풀이<br/>

```python
m = int(input())
n = int(input())
lis = []
lis2 = []

for i in range(m, n+1):
    for j in range(1, i+1):
        if i/j - i//j == 0:
            lis.append(i)
        else:
            pass
for i in range(m, n+1):
    if lis.count(i) == 2:
        lis2.append(i)
    else:
        pass
if len(lis2)>0:
    print(sum(lis2))
    print(min(lis2))
else:
    print(-1)
    
```
