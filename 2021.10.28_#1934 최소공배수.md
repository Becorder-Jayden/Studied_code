## # 1934 최소공배수

<br/>

멘토링 Tip

일반적인 풀이로 진행할 경우 런타임 에러가 나도록 의도한 문제이다.

1. 최소공배수는 두 수의 곱을 최대공약수로 나누면 되는 성질 이용
2. 어떻게 하면 최대공야수를 빠르게 구할 것인지 고민

결론적으로 *유클리드 호제법을 사용해 풀이해야 한다

<br/>

유클리드 호제법을 활용한 풀이

```python
def gcd(a, b):
    while b > 0:
        a, b = b, a % b
    return a

def lcm(a, b):
    return a * b // gcd(a,b)
```

<br/>
